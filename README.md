Project 2: Intel Sustainability Journey

An interactive, responsive timeline that highlights Intel’s sustainability goals and long-term environmental commitments.

This project explores layout structure, user interaction, and responsive design by transforming static sustainability information into an engaging visual experience.

Overview

The goal of this project is to design a timeline-based webpage that:

Presents Intel’s sustainability milestones clearly and visually

Incorporates hover effects and smooth transitions

Adapts seamlessly to both desktop and mobile devices

Maintains a clean, modern layout

Features

Interactive timeline interface

Responsive layout using flexible design principles

CSS hover effects and transitions

Organized, accessible content structure

Tech Stack

HTML

CSS

Media Queries for responsiveness

GitHub Codespaces for development

Getting Started

To run this project:

Launch a GitHub Codespace from the repository.

Make changes locally within Codespaces.

Commit and push changes regularly to avoid losing progress.
